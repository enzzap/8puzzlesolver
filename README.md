# CS170 N-puzzlesolver
Program made in Python 3 which solves the n-Puzzle problem with Uniform Cost Search, A* search using Misplaced tile heuristic and A* search using Manhattan distance heuristic.
# About
This was made as the first Project for my CS170 Class with Dr. Eamonn Keogh.
# How to Run
just run the py file in a python 3 environment
